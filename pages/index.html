PolyCloud OpenAlpha

polycloud is a free opensource cloud storage is based off lolisafe on github.com with almost no thirdparty software and resources. You can access and share files by uploading a single file, by using one of many methods (zip, gzip, mkv, mp3, m4a, etc) as well as by simply dragging-and-dropping folders on it. It offers real-time collaboration for file uploads, no files sent to any cloud storage services, and good support for multiple devices for accessing files. It has strong anti-malware protection, and allows for private cloud storage sharing and password-less sharing between smartphones, tablets and Personal Computers

in order to share files:

  1. copy the link
  2. put the link in a browser
  3. some files will automatically download, for example zips
